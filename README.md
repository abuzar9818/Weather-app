# Weather App

A responsive weather application that provides real-time weather information for cities around the world using the RapidAPI Weather API.

## Features

- Search for weather information by city name
- View detailed weather data including:
  - Current temperature and feels-like temperature
  - Temperature min/max
  - Humidity and atmospheric pressure
  - Visibility distance
  - Sea level and ground level pressure
- Pre-loaded weather data for popular cities (Boston, Lucknow, Beijing)
- Responsive design that works on desktop and mobile devices
- Clean and intuitive user interface with Bootstrap styling

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5
- Font Awesome icons
- RapidAPI Weather API

## How to Use

1. Open `index.html` in a web browser
2. Enter a city name in the search bar at the top
3. Click "Search" or press Enter to view weather data
4. Browse pre-loaded weather information for Boston, Lucknow, and Beijing at the bottom of the page

### Weather Data Explanation

- **Temperature**: Current temperature in Kelvin
- **Feels Like**: Perceived temperature considering humidity and wind
- **Humidity**: Percentage of moisture in the air
- **Pressure**: Atmospheric pressure in hPa
- **Visibility**: Distance you can see clearly
- **Sea Level/Ground Level**: Pressure measurements at different altitudes

## API Information

This application uses the [RapidAPI Weather API](https://rapidapi.com/weatherapi/api/weather-api138/) to fetch real-time weather data.
