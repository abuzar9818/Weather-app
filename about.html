About This Weather App

This weather application provides real-time weather updates for cities around the world. It fetches data such as temperature, humidity, pressure, visibility, and more using the RapidAPI Weather API.

Built using HTML, Bootstrap, and JavaScript, this app offers a user-friendly interface and is designed for quick weather lookups. Whether you're planning a trip or just checking your local weather, this app has you covered.

You can search for any city and get detailed weather information instantly. It also includes data like sea level, ground level, and feels-like temperature.
